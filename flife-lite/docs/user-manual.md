# FLIFE Lite User Manual

## Workflow

1. Activate a trial or paid license.
2. Create or open a project.
3. Import CSV, XLSX, or TXT stress history data.
4. Select or edit a material.
5. Configure fatigue method, mean stress correction, cycle counting, and safety factor.
6. Run analysis.
7. Review life prediction, damage rate, cycle histogram, and SN curve.
8. Generate a PDF report.

## Supported Analysis Methods

- Rainflow counting.
- Dirlik spectral method.
- Narrowband spectral method.
- Tovo-Benasciutti spectral method.

## Offline Use

After activation, the desktop client stores an encrypted token and can continue in offline mode until the configured grace period expires.
